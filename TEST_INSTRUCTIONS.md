# README

## Setup Test Integrations
Execture unit test with package unitest: `py -m test_calculator`
## Test Code Coverage

### Generate Code Coverage:
`py -m coverage run .test_calculator.py`

### Get All Code Coverage Report: 
`py -m coverage report --omit="test*"`
